# Seed Data Directory

This directory contains JSON files for seeding the MongoDB database with initial insurance data.

## Files

- `seed_insurers.json` - Insurance company information
- `seed_vehicles.json` - Vehicle catalog
- `seed_plans_template.json` - Template for insurance plans (generated by script)
- `seed_signals_template.json` - Template for service quality signals (generated by script)

## Data Generation

The actual plans and signals data will be generated using the `generate_seed_data.py` script to create realistic combinations of:

- Insurance plans for different vehicle types and regions
- Service quality signals (claim TAT, satisfaction, renewal rates, etc.)

## Loading Data

Use the Flask service data loader:

```python
from database import get_db
from utils.data_loader import DataLoader

db = get_db()
loader = DataLoader(db)

# Load insurers
loader.load_from_json_file('data/seed_insurers.json', 'insurers')

# Load vehicles
loader.load_from_json_file('data/seed_vehicles.json', 'vehicles')

# Load plans (after generation)
loader.load_from_json_file('data/seed_plans.json', 'plans')

# Load signals (after generation)
loader.load_from_json_file('data/seed_signals.json', 'signals')
```

## Data Schema

### Insurers
```json
{
  "name": "string",
  "region_codes": ["string"],
  "website": "url",
  "phone": "string",
  "description": "string"
}
```

### Plans
```json
{
  "insurer_id": "ObjectId",
  "plan_name": "string",
  "vehicle_types": ["string"],
  "region_codes": ["string"],
  "premium_annual": float,
  "idv": float,
  "add_ons": ["string"]
}
```

### Signals
```json
{
  "plan_id": "ObjectId",
  "claim_tat_days": int,
  "claim_approval_rate_pct": float,
  "csat_score": float,
  "renewal_rate_pct": float,
  "complaint_ratio": float
}
```
